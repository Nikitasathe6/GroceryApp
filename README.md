# 📖 Overview

The **Grocery Management System** is a Java-based web application developed to simplify grocery store operations. It enables administrators to efficiently manage products, inventory, customer orders, and billing through a user-friendly interface.

This project was developed as part of my **Bachelor of Science in Computer Science** to strengthen my practical knowledge of Java Full Stack Development.

---

# ✨ Features

- 🔐 Secure Admin Login
- 📦 Product Management (Add, Update & Delete)
- 📊 Inventory Management
- 🛒 Customer Order Processing
- 💳 Billing System
- 🗄️ PostgreSQL Database Integration
- 📱 Responsive User Interface
- ⚡ Fast and Efficient Operations

# 🚀 Installation Guide

### Clone Repository

```bash
git clone https://github.com/Nikitasathe6/GroceryApp.git
```

### Open Project

Import the project into **Eclipse IDE**.

### Configure Database

- Install PostgreSQL
- Create the required database
- Update database credentials in `DBConnection.java`

### Run Application

- Deploy the project on **Apache Tomcat**
- Start the server
- Open the application in your browser
